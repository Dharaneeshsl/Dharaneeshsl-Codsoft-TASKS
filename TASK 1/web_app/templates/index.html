How it Works

This application uses machine learning models trained on movie plot summaries to predict genres. The system employs:

  • TF-IDF Vectorization: Converts text into numerical features
  • Text Preprocessing: Cleans and normalizes the input text
  • Multiple Classifiers: SVM, Logistic Regression, Naive Bayes, and Random Forest
  • Confidence Scoring: Provides prediction confidence levels

Tips for Better Predictions

  • Provide detailed plot summaries (50+ words)
  • Include key plot elements, characters, and themes
  • Mention genre-specific keywords (e.g., "magic" for Fantasy, "detective" for Crime)
  • Avoid spoilers but include enough context
